Stereo Generator Overview

Stereo Generator consists of the following modules:

Depth Refinement — corrects inaccurate depth area borders by attaching them to object edges in the original image; also sharpens blurry depth map borders
(After Effects|Nuke)
Depth Normalization — stabilizes a depth map and normalizes frame depth maps to the same range of values
(After Effects|Nuke)
Stereo Generator — main 2D+Depth-to-stereo conversion module
(After Effects|Nuke)
Stereo Ghost Busting — compensates for stereo crosstalk between left and right stereomate images from input stereo video
(After Effects|Nuke)
Background Reconstruction — high quality reconstruction of unknown areas
(After Effects|Nuke)
Matting — special processing of fuzzy edges of objects like hairs, fur and foliage
(After Effects|Nuke)

Features

  • True 32 bit color depth
  • Arbitrary parallax
  • Automatic depth map and stereo correction for quality improvement
  • Processing speed is about 3 fps for HD 1080p content

Technical Requirements

  • Adobe After Effects:
    • Adobe After Effects CS5.5 or later
    • Microsoft Windows 64-bit environment
    • At least 2GB of RAM; 8GB or more recommended
  • Nuke:
    • Nuke 6.2 x64 or later
    • Microsoft Windows 64-bit environment
